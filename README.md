# 🎓 Projeto de Extensão: Primeiro Passo para o Mercado de Trabalho


Este projeto é uma plataforma interativa desenvolvida para apoiar jovens em busca do primeiro emprego ou transição de carreira. O foco central é a otimização estratégica do perfil no **LinkedIn**, transformando-o em um verdadeiro ímã de oportunidades.

## 🚀 Funcionalidades

- **Trilha de Aprendizado em Vídeo:** Curadoria de tutoriais essenciais para sair do zero ao perfil campeão.
- **Anatomia do Perfil Perfeito:** Guias visuais detalhados sobre posicionamento, foto, headline e sobre.
- **Biblioteca de Artigos:** Acesso direto a materiais de instituições de ensino renomadas sobre carreira.
- **Interface Tech:** Design moderno em Dark Mode com foco em UX para uma experiência imersiva.

## 🛠️ Tecnologias Utilizadas

- **HTML5** e **CSS3** (Variáveis CSS e Flexbox/Grid).
- **Google Fonts** (Fonte Inter).
- **Design:** Inspirado em Dashboards de alto desempenho (Glassmorphism & Dark Tech).
